Narrative

This project uses the following technologies and libraries:

  • Server:
    • GCP -- as the cloud platform
    • MySQL 5.7 -- as the database
    • Werkzeug 0.15.5 -- as the server
    • Python 3.7 -- as the primary language
    • Flask 1.0.2 -- as the application framework
    • Flask-SQLAlchemy 2.4.0 -- as an object-relational mapper
    • pytz 2019.2 -- to manage timezones
  • Web interface:
    • jQuery 3.4.1 -- for DOM queries and ajax calls
    • Bootstrap 4.3.1 -- to provide theming and base styles
    • lodash 4.17.15 -- to better handle data structures
  • Mobile interface:
    • Kotlin -- for the primary language
    • Gradle -- as a task runner
    • OkHttp -- to handle server calls

Using the web interface

The user may log in at the landing page by entering their email address and password.

If they have not yet, they may register using their details.

Upon logging in, the user sees a list of the garages and the available spots at these garages. They may click on a card to book a spot.

Once a spot is reserved, the user is informed about the reservation and given a choice to occupy their spot.

A user that is currently occupying a spot may check out any time.
